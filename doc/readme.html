Moorea Biocode LIMS Plugin

The Biocode software comprises everything you need to manage your lab and sequence analysis workflows. See the official wiki at http://software.mooreabiocode.org for more information on how to use the plugin.

Release notes are available at http://software.mooreabiocode.org/index.php?title=Release_Notes

Getting Help

A great source of help can be found at the discussion forum at http://connect.barcodeoflife.net/group/lims. Or if you have any questions or suggestions that you do not want to be public you can email support@mooreabiocode.org.
